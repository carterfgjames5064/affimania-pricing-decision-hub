## Overview

Affimania gives teams and solo users a browser-first way to inspect pricing data with AI support. Instead of depending on a desktop application, it uses a web app workflow to help you review, compare, and interpret pricing information directly in the browser.

As a compact static web application, it is simple to host and quick to open in any standard browser. That makes it a practical option when you need a clean interface for pricing analysis, rapid side-by-side comparison, and a direct route from raw pricing data to more informed choices.
